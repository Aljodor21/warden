Acerca de warden

Qué es

Base opinionada y reutilizable para montar y mantener tu propio servidor casero (homelab): instalación desde cero, backup y restauración, dashboard y CI/CD para tus propios proyectos — en Debian/Ubuntu o Arch. Todo tiene dos caminos equivalentes: la consola (warden ...) o este panel web — ninguno es "el oficial".

Los datos se respaldan cifrados, versionados y deduplicados con restic. La configuración vive como código y es reproducible — si toca recuperarte o migrar a otro sistema operativo, el camino ya está trazado.
